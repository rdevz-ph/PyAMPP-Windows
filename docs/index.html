Project Overview

PyAMPP is an advanced orchestration utility that automates the lifecycle of a WAMP stack. It eliminates the friction of manual configuration by dynamically generating environment-specific parameters at runtime, ensuring a "it just works" experience for local web development.

This software is developed for personal use and internal development purposes. The source code is not available for public distribution or external contribution.

Detailed Features

Automated Setup Wizard

One-click deployment that handles remote asset retrieval, integrity validation, and structured extraction of Apache, MySQL, PHP, and phpMyAdmin.

Granular Service Control

Independent process management for Apache and MySQL. Features including real-time PID monitoring, port availability checks, and unified stack orchestration.

Intelligent PHP Configuration

Auto-generates php.ini with pre-configured extensions (mysqli, mbstring, openssl, curl, pdo_mysql, zip, gd, fileinfo, intl, exif, etc.) mapped to absolute local paths.

Secure MySQL Initialization

Automates the --initialize-insecure routine for new MySQL data directories, ensuring immediate accessibility while maintaining local portability.

Global PATH Integration

Integrated PowerShell automation to register the portable PHP binary within the Windows System PATH, enabling CLI usage (composer, artisan, etc.) globally.

phpMyAdmin Orchestration

Automatic deployment to Apache htdocs with dynamic blowfish_secret generation and pre-configured local-only access controls.

LAN Exposure Toggle

Switch between private loopback (localhost) and local network visibility. Automatically detects subnet masks to configure Apache access controls safely.

Real-time Log Aggregator

Captures and streams stdout and stderr from underlying service processes into a centralized dashboard for rapid diagnostic troubleshooting.

Managed Components

PyAMPP targets specific, high-performance distributions of standard web components:

  • Apache HTTP Server: VS17 Win64 Binaries from Apache Lounge (configured with Windows-specific performance optimizations like AcceptFilter none).
  • MySQL Community Server: 8.0.x x64 distributions with custom my.ini templating.
  • PHP: Thread Safe 8.3.x distributions, automatically integrated as an Apache module.
  • phpMyAdmin: Latest multi-language releases with automated configuration mapping.

Technical Specifications

  • Core Logic: Python 3.x utilizing the subprocess API for detached process execution.
  • GUI Framework: CustomTkinter for a modern, High-DPI aware Windows 10/11 native look.
  • Networking: Synchronous socket-based port monitoring and automated subnet detection.
  • Configuration: JSON-based persistent state management for user preferences and environment paths.
  • Process Isolation: Detached process trees to ensure services remain stable independently of the GUI state.

System Architecture

Designed with a strict separation of concerns:

  • Core Layer: Encapsulates binary management, configuration templating (Apache httpd.conf, MySQL my.ini, PHP php.ini), and low-level process control.
  • GUI Layer: Modular interface components (Dashboard, Setup Wizard, Settings, Global PHP Setup, and Log Aggregator).
  • Automation Layer: PowerShell scripts for system-level integrations and automated environment pathing.
